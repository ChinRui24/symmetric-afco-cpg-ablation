% We conducted systematic ablation studies to evaluate the contribution of each 
% component in our symmetric AFCO CPG framework. As shown in Table~\ref{tab:ablation_main}, 
% the full model achieved the best performance across all metrics, with a phase 
% synchronization error of 0.155 rad and disturbance recovery time of 4.77 s.
% 
% Notably, removing the symmetric PRC mechanism resulted in the most significant 
% performance degradation, increasing phase synchronization error by 106\% and 
% recovery time by 110\%. This demonstrates that symmetric PRC is the most critical 
% innovation in our framework.
% 
% Table~\ref{tab:component_contribution} quantifies the contribution of each component. 
% The symmetric PRC alone contributes to 51.5\% improvement in phase synchronization, 
% followed by GRF weighting (33.6\%) and adaptive coupling (14.7\%).
% 
% Statistical significance tests (Table~\ref{tab:statistical_test}) confirm that 
% the performance differences between the full model and all ablated configurations 
% are statistically significant ($p < 0.05$), with the most significant difference 
% observed when removing symmetric PRC ($p < 0.001$).
